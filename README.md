# Agendar Citas App

Proyecto Universitario

Universidad de la Costa 'CUC'

Lenguaje: PHP

Framework usado: CodeIgniter

## Integrantes:
  - Manuel Perez
  - Jerson Orozco

## Presentado a:

Harol Combita

Esya aplicacion para agendar citas consta de 3 modelos:
- Pacientes
- Medicos
- Citas

## Campos de pacientes
  - Nombre
  - Apellido
  - Direccion
  - Cedula

## Campos de medicos
  - Nombre
  - Apellido
  - Direccion
  - Cedula
  - especialidad

## Campos de cita
  - Fecha
  - Hora
  - Paciente
  - Medico
