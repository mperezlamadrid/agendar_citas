# Agendar Citas App

Proyecto Universitario

Lenguaje: PHP
Framework usado: CodeIgniter

Esya aplicacion para agendar citas consta de 3 modelos:
- Pacientes
- Medicos
- Citas

## Campos de pacientes
  - Nombre
  - Apellido
  - Direccion
  - Cedula

## Campos de medicos
  - Nombre
  - Apellido
  - Direccion
  - Cedula
  - especialidad

## Campos de cita
  - Fecha
  - Hora
  - Paciente
  - Medico
